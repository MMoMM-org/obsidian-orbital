# Commands Reference

Orbital contributes a single command to the Obsidian command palette. Most of Orbital's
functionality lives inside the pane itself (see [Usage](usage.md)) and its
[settings](settings-reference.md), rather than as separate commands.

## Commands

| Command | Description | Preconditions | Default hotkey |
|---------|-------------|---------------|----------------|
| `Orbital: Open` | Opens the Orbital pane in the right sidebar, or reveals it if it's already open. | None | None |

## Notes

- **No default hotkey.** Assign one under **Settings → Hotkeys** (search for "Orbital")
  if you open the pane often.
- You can also open the pane by clicking the Orbital **status-bar item** (when enabled
  in Settings → Orbital), which opens it on the Relations tab.
